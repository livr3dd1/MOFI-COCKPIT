# MOFI-COCKPIT
Mofi cockpit!, a little project created to manage your Arch Linux System easily and directly, things like Search function, Processes iterating, etc are included
Here  you can Iterate processes, Check Wifi connection, update your system, Search files, Shutdown or Rebooot your computer, Reset your wifi, Or see available Wifi Connections

# Installation
When you download Mofi, Go to the folder where you downloaded it, then do
'chmod +x installer.sh'
And then:
'sudo ./installer.sh'
And wait for it to Finish, thats all, enjoy it!
